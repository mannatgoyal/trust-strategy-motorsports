# Changelog

All notable changes to the **F1 Strategy Engineer Toolkit** will be documented in this file.

---

## [1.0.0] - 2026-07-16

### Added
*   Thermodynamic tyre compound wear model supporting Soft, Medium, and Hard profiles.
*   Non-linear fuel weight penalty solver using quadratic decay.
*   Sigmoid-based overtaking probabilities and dirty air wake drag penalties.
*   Bayesian Safety Car posterior probability profiler.
*   Multi-variable joint correlated Monte Carlo simulator.
*   Interactive live Race Replay timeline auditor.
*   Central configuration file (`race_config.yaml`) mapping track profiles.
*   GitHub Actions CI workflow for test automation.
*   Descriptive docstrings and full type annotations.
